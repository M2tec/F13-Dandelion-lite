Module 101: Introduction

Welcome to Dandelion-lite PBL 2025!

How this course works:

  • All Project-Based Learning courses consist of a series of modules.
  • Each module begins with a list of Student Learning Targets (SLTs).
  • For each SLT, there is a lesson that introduces some important ideas. Usually, these lessons show you how to do something.
  • We believe that learning by doing is the most effective way to learn. That's why we will introduce several different projects throughout this course.

Welcome to the first lesson on Dandelion-lite server API’s, focusing specifically on how to setup your own node. This lesson will guide you through configuring your own server to run decentralized API’s.

Prerequisites

This is a basic course on setting up you own server. The purpose of Dandelion-lite is to make setting up your server as easy as possible. To start this course mare sure you are already familiar with the following:

  • Ubuntu server basic maintenance. Installing packages
  • Bash command line interface usage.
  • Remote server login using ssh
