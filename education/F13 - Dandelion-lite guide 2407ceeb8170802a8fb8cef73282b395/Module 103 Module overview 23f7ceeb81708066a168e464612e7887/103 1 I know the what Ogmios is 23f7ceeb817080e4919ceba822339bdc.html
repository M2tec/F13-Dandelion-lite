103.1 I know the what Ogmios is

Cardano Ogmios is a lightweight bridge that makes Cardano-node accessible to developers outside of Haskell:

🔧 What is Ogmios?

  • Protocol translator: Ogmios is a JSON/RPC‑over‑WebSocket interface that sits on top of a running cardano‑node. It speaks Cardano’s compact "Ouroboros mini‑protocols" in Haskell and exposes them as standard WebSockets and JSON calls that most programming environments can understand Reddit+15Ogmios+15Demeter Docs+15.
  • Allows non‑Haskell developers to interact directly with a live Cardano node—no need to write Haskell or use exotic network protocols Reddit+2Ogmios+2Reddit+2.

🧑‍💻 What it’s used for

Ogmios is ideal for low-level integrations, enabling applications such as:

  • Wallet back-ends
  • Blockchain explorers
  • DApps and smart‑contract applications
  • Real‑time dashboards or bots monitoring on-chain activity

Essentially, anything that requires access to protocol-level details like UTXOs, block sync, transaction submission, stake distribution etc. Demeter+4Built on Cardano+4Ogmios+4GitHub.

⚙️ How it works

  • A developer runs a cardano-node locally or connects to a relay node.
  • Ogmios translates native mini-protocols (chain sync, ledger queries, transaction submission) to JSON/RPC messages over WebSockets (or HTTP) Reddit+15GitHub+15GitHub+15.
  • It supports detailed actions such as:
    • Syncing chain progress (blocks and headers)
    • Querying ledger state, UTXOs, stake pool data, rewards, protocol parameters
    • Submitting transactions with enhanced error feedback
    • Monitoring mempool, tips, network health, runtime statistics GitHubGitHub+3GitHub+3Cardano Developer Portal+3

🧠 Why use Ogmios?

  • Lightweight alternative to full node‑plus‑cardano‑db‑sync setups; easier, faster to deploy and works close to the metal Demeter+3Ogmios+3Reddit+3.
  • Ideal if you need every protocol detail (as opposed to higher‑level services like GraphQL or Blockfrost) GitHub+7Ogmios+7Reddit+7.
  • Deployable alongside relay nodes for multi-tenant, real‑time service endpoints (e.g. block explorers, stake monitor dashboards) Reddit+4Reddit+4GitHub+4.

📦 Example usage

  • Run both cardano-node and Ogmios via the Docker Compose setup from the official repo — includes a real-time dashboard at localhost:1337 and health metrics endpoint Demeter+15Cardano Developer Portal+15GitHub+15.
  • Connect through community SDKs (TypeScript/JavaScript, Python, Go) to send JSON/RPC requests and receive responses Built on CardanoDemeter.
  • Example use: submit a transaction or query UTXO balances by sending JSON-RPC messages authenticated and processed over WebSocket Demeter.
