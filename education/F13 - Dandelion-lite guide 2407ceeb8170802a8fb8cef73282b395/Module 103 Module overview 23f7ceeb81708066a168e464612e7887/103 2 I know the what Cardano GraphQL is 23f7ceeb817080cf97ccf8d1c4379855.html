103.2 I know the what Cardano GraphQL is

Cardano GraphQL is an open-source GraphQL API layer for querying data from the Cardano blockchain. It provides a flexible and powerful way to access on-chain data using the GraphQL query language, which is widely adopted in modern web development.

It runs on top of a PostgreSQL database that is synchronized with the Cardano blockchain (usually via cardano-db-sync), exposing the data in a structured, typed format.

üì¶ Main Components

  1. cardano-graphql

    The main GraphQL server that exposes the blockchain data.

  1. cardano-db-sync

    A backend tool that syncs data from the Cardano node into a PostgreSQL database.

  1. PostgreSQL

    The relational database where blockchain data is stored for querying.

  1. Hasura (optional)

    Sometimes used as a dynamic GraphQL engine over PostgreSQL, though not always included in official setups.

üß© What Can You Query?

You can query detailed blockchain data, such as:

  • Blocks (height, slot, hash, timestamp)
  • Transactions (hash, inputs/outputs, fees, metadata)
  • Addresses (UTxOs, balances)
  • Stake pools (delegations, rewards, owners)
  • Epochs and slots
  • Asset data (native tokens, metadata)
  • Metadata associated with transactions

Example query:

{
  transactions(limit: 5) {
    hash
    block {
      number
      slotNo
    }
    inputs {
      address
      value
    }
    outputs {
      address
      value
    }
  }
}
