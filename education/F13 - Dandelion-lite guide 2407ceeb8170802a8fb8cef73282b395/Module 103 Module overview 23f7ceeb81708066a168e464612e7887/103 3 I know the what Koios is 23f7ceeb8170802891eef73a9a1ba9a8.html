103.3 I know the what Koios is

Koios is a lightweight and scalable API service for the Cardano blockchain, designed to make it easier for developers and applications to query on-chain data without running a full node.

🔍 Key Features of Koios

  1. RESTful API Access

    Koios exposes Cardano data through a standard REST API, making it easy to integrate with any backend or frontend application without needing GraphQL or custom Cardano infrastructure.

  1. No Need to Run Your Own Node

    It removes the need to run a full Cardano node or sync the blockchain. This is helpful for developers who just want access to blockchain data (e.g. UTxOs, transactions, blocks, stake info, etc.).

  1. Lightweight & Efficient

    It's built for performance and low resource consumption, suitable for use on smaller servers or edge applications.

  1. Supports Multiple Networks

    Koios supports mainnet, preview, preprod, and other Cardano testnets.

  1. Open Source

    Koios is open source, so anyone can deploy their own instance, modify the code, or contribute to the project.

📦 Common API Endpoints

  • /tip – Get the current tip of the blockchain
  • /address_info – Get info about a specific address (e.g., balance, UTxOs)
  • /tx_info – Fetch full data of a transaction
  • /asset_info – Get details about a native token
  • /epoch_info – Info about current or past epochs
  • /pool_list – Stake pool information
  • /account_info – Stake account details
