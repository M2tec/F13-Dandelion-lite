102.2 I know how to setup our node server

Great you have installed your operating system. You are now ready to install your first dandelion nodes. Let’s start with Cardano preprod.

1. Change swap file size

The server is running bigger RAM than most machines. Check your swap space. You will need more swap space if you have more RAM. In the case of Ubuntu Desktop we change from default (8GB) to 40G

swapon --show
sudo swapoff -v /swap.img
sudo rm -rf /swap.img
sudo fallocate -l 40G /swap.img
sudo chmod 0600 /swap.img
sudo mkswap /swap.img
sudo swapon
sudo reboot now

3. Configuring reboots for security updates

If for some reason the package is not present, it can be installed with the following command:

sudo apt install unattended-upgrades

The server is always running. In some cases a reboot is required for updates to take effect. Because Dandelion-lite is a group of servers, it is generally safe to do this automatically. 

sudo nano /etc/apt/apt.conf.d/50unattended-upgrades

Change the following lines

// Automatically reboot WITHOUT CONFIRMATION if
//  the file /var/run/reboot-required is found after the upgrade
Unattended-Upgrade::Automatic-Reboot "True";

// Automatically reboot even if there are users currently logged in
// when Unattended-Upgrade::Automatic-Reboot is set to true
//Unattended-Upgrade::Automatic-Reboot-WithUsers "true";

// If automatic reboot is enabled and needed, reboot at the specific
// time instead of immediately
//  Default: "now"
Unattended-Upgrade::Automatic-Reboot-Time "02:00";

In this case when needed the server will reboot at 2 o’clock at night.

4. Cloning Dandelion-lite github

Cardano can be configured to connect to different networks for example testing. Dandelion-lite comes with two configurations out of the box. Preprod and Mainnet.

Let’s use git to clone the Dandelion-lite repository. Every network requires it’s own docker compose project. So this is a separate git clone. Clone into a folder you put your projects. For example ~/src ( ~ is shorthand for your home folder)

git clone https://github.com/GameChangerFinance/dandelion-lite.git dandelion-preprod

Here we clone the Dandelion-lite repo into the dandelion-preprod folder.
