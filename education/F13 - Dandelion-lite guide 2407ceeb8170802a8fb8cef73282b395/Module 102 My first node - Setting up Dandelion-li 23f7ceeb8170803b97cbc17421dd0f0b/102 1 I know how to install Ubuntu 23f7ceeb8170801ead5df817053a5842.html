102.1 I know how to install Ubuntu


This guide will walk you through installing
Ubuntu 24.04 LTS (Noble Numbat) on your computer using a USB stick.

Requirements

  • A computer with at least:
    • 4 GB RAM
    • 25 GB free disk space
  • A USB stick (≥ 4 GB)
  • Ubuntu 24.04 LTS ISO

    LINUX TYPEVERSIONLINKDATE
    Server [LTS]24.04click here12 April 2025
    Desktop [LTS]24.04click here12 April 2025
  • A tool to write the ISO to USB (e.g., Rufus, balenaEtcher, or GNOME Disks)

1. Create a bootable drive

You can use a thumb-drive, usb-drive, external harddrive, SD-card or even a disc if your server and support system both have disc drives.

***NOTE***
Any data on the media being used for this process will be wiped of data
  • MacOS: install balenaEtcher and follow instructions
  • Windows: install UNetbootin and follow instructions
  • Linux: install UNetbootin and follow instructions

2. Boot from USB

  1. Insert the USB stick into your computer
  1. Reboot and enter your BIOS/UEFI settings (usually by pressing F2, F10, F12, or DEL)
  1. Change the boot order to boot from USB first
  1. Save and exit

Your machine should now boot into the Ubuntu installer.

3. Install Ubuntu 24.04 LTS

After booting into the installer:

Step 1: Choose Language

Select your preferred language and click Try or Install Ubuntu.

Step 2: Keyboard Layout

Choose your keyboard layout and continue.

Step 3: Network (Optional)

Connect to Wi-Fi (optional — updates can be installed later).

Step 4: Installation Type

Choose one of the following:

  • Erase disk and install Ubuntu – wipes the entire disk
  • Install alongside another OS – dual-boot setup
  • Something else – manual partitioning
⚠️ Back up your data before selecting Erase disk!

Step 5: Choose Installation Options

  • Select Normal or Minimal installation
  • Enable Download updates and Install third-party software if needed

Step 6: Timezone

Select your location on the map to set your timezone.

Step 7: Create a User

Fill in your name, username, password, and computer name.

4. Finish Installation

  • Ubuntu will copy files and install the system
  • Once done, remove the USB when prompted
  • Click Restart Now

5. First Boot

Log into your new Ubuntu 24.04 LTS system using the credentials you created. You’re now ready to begin using Ubuntu!

6. Install SSH

Install OpenSSH server to allow for remote system administration.

sudo apt install openssh-server
  1. When install is completed do ip -br a to get the ip address of your newly installed server
  1. From you local system. So not on your server do ssh-copy-id USER_NAME@SERVER_IP This will copy your ssh keys to the server to allow easy login
  1. Do ssh USER_NAME@SERVER_IP
  1. It is also possible to create a shorthand for ssh login. This is done in the ~/.ssh/config file. And looks like this.

    Host dev
    HostName dev.example.com
    User john
    Port 2322

    Now when you type ssh dev, the ssh client will read the configuration file and use the connection details that are specified for the dev host:

    ssh dev

Troubleshooting

  • No boot option appears?
    • Make sure Secure Boot is disabled in UEFI
    • Double-check the USB boot order
  • The Ubuntu installer crashed.
    • Boot the live image into: Ubuntu (safe graphics)
