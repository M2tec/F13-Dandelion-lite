101.2 I understand what a server API is

A server API (Application Programming Interface) is a way for different software applications to communicate with a server, typically over the internet.

In simple terms:

A server API allows a client (like a web app, mobile app, or other system) to send requests to a server, and the server responds with data or performs actions.

For example:

Imagine a weather app on your phone. When you open it:

  • The app sends a request to a weather API on a server.
  • The server responds with data like temperature and humidity in your area.

Key Concepts

ConceptMeaning
ServerA computer running a program that listens for and responds to requests
ClientThe app, browser, or system making the request
APIA set of rules for how to interact with the server
RequestA message sent to the server (usually HTTP methods like GET, POST)
ResponseData sent back from the server, usually in JSON or XML format

Common Types of Server APIs

  • REST API: Uses HTTP methods (GET, POST, PUT, DELETE) and URLs.
  • GraphQL API: Client specifies exactly what data it wants.
  • gRPC: Uses protocol buffers for fast, compact communication.

Example of a REST API call

Client request:

GET /users/123 HTTP/1.1
Host: api.example.com

Server response:

{
  "id": 123,
  "name": "Alice",
  "email": "alice@example.com"
}
